# 🌌 Reference Finder 

Reference Finder is an AI-powered visual mood boarding tool designed for creators, designers, and visual directors. Type in an abstract concept, and let machine learning isolate core aesthetic motifs and generate real-time design references.

![UI Overview](./public/screenshot.png)

## ✨ Core Features

*   **⚡ AI-Driven Keyword Breakdown:** Uses Groq's high-speed inference to decode short, abstract prompts into precise aesthetic queries.
*   **🌐 Interactive 3D Sphere:** Seamless canvas with responsive drag gestures and custom automated physics.
*   **📋 Cross-Platform Functionality:** Effortless clipboard operations for continuous workflows.

## 🛠️ Requirements
*   **Next.js 15+**
*   **Node.js 18+**

## 🚀 Environment
Add the appropriate keys:
```env
GROQ_API_KEY=your_key_here
UNSPLASH_ACCESS_KEY=your_key_here
```

## ⚙️ Installation
```bash
npm install
npm run dev
```
